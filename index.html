Bring Your Own Diet



Project One was a team effort called Bring Your Own Diet. This food application highlighted the best recipes and restaurants in every State with a bonus feature of searching for recipes all while being able to save any favorites logged in. I was a back-end developer in this project but also designed some of the front-end work. We used technologies such as HTML, CSS, Bootstrap, Animate, jQuery, Edamam API, Opentable API, and Firebase.

Ventura Burgers



In this appplication called Ventura Burgers, I showcase the abilities of a full stack application. From selecting data from a database to a user being able to enter a name of a burger that will appear on screen shortly after. After you may update that data with a simple click of a button. This applicationn was fun to design, implement and build using technologies like HTML, CSS, jQuery, Node.js, Express.js, Express-Handlebars, mySQL. I deployed this project with Heroku and a JawsDB add-on.

Friend Finder



Friend Finder is a again showcasing the abitlity to take in simple data from a user and manipulate it to do something more meaningful like matching people. This application lets users fill out a survey about their interests and behind the scenes those question answers are used to match them with a friend. For this application I used technologies like HTML, CSS, Bootstrap, jQuery, Node.js, and Express.js.
