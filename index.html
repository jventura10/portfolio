Better Health




Better Health is a medical application that provides easy to use features for a patient or someone in a medical staff. Including appointment scheduling, messaging, and handling medical records. This project showcases multiple components interacting to give a more complete service to users.

I was Back-End Lead to the project and primarily handled user authentication and database operations but helped connect these components to front-end and routes that needed these features. We used NodeJS and ExpressJS for setting up the server and routing. For authentication we used PassportJS and Bcrypt to help hash and store passwords, as well as easing the process of logging in and signing up. We used sequelize to access our MySQL Database which was a good choice for associations and our schema demands. In the front-end UIKit was used to style the site and jQuery for handling AJAX calls and listening for user interactions. The twitter API was used as well to give the user another reason to love our site while in the waiting room.

Bring Your Own Diet




This food application highlighted the best recipes and restaurants in every State with a bonus feature of searching for recipes all while being able to save any favorites logged in. I was a back-end developer in this project but also designed some of the front-end work. We used technologies such as HTML5, CSS, Bootstrap, and Animate to style the site. jQuery was used for listening for user interactions. The Edamam API and Opentable API were used to find recipes and restaurants, respectively, because of their large data sets. Firebase was used to ease the login and sign-up process, this was also a good fit because of the noSQL style of Firebase that fit our schema where each user's data will vary greatly.

Ventura Burgers




In this appplication called Ventura Burgers, I showcase the abilities of a full stack application. From selecting data from a database to a user being able to enter a name of a burger that will appear on screen shortly after. After you may update that data with a simple click of a button. This applicationn was fun to design, implement and build using technologies like HTML, CSS, jQuery, Node.js, Express.js, Express-Handlebars, mySQL. I deployed this project with Heroku and a JawsDB add-on.
